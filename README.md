# Sales Forecasting Using Machine Learning

## Project Overview

This project aims to forecast future sales using historical sales data. Machine Learning techniques were applied to analyze patterns and predict future sales performance.

## Objective

* Analyze sales data
* Identify important factors affecting sales
* Build a forecasting model
* Evaluate prediction performance

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Jupyter Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building using Linear Regression
6. Sales Prediction
7. Model Evaluation

## Results

* MAE: 6857.54
* RMSE: 9524.55
* R² Score: 0.868

The model successfully explains approximately 86.8% of the variation in sales data.
